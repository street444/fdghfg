# ⭐ Бухгалтерія — Система управління документами

Сучасна, сучасна платформа для управління документами, груповою співпрацею та бухгалтерськими звітами. Повністю працює **офлайн** у браузері.

## 🎯 Основні можливості

✅ **Реєстрація та вхід** — без сервера, все в localStorage  
✅ **Групи** — публічні та приватні, з управлінням учасниками  
✅ **Документи** — створення, редагування, видалення  
✅ **Редактор** — вбудований редактор з таблицями та форматуванням  
✅ **Файли** — завантаження та зберігання файлів (Base64)  
✅ **Пошук та фільтри** — швидкий пошук за датою та назвою  
✅ **Автозбереження** — кожні 10 секунд  
✅ **Cookies** — опція "Запам'ятати мене"  
✅ **Повністю адаптивний** — працює на мобільних і ПК

## 🚀 Як запустити

### Локально у браузері
1. **Завантажте проєкт** на комп'ютер
2. **Відкрийте `index.html`** у браузері (або запустіть локальний сервер):
   ```bash
   # Якщо маєте Python 3:
   python -m http.server 8000
   # Або Node.js (http-server):
   npx http-server
   ```
3. **Перейдіть** на `http://localhost:8000`

### На GitHub Pages
1. **Завантажте репозиторій** на GitHub
2. **Перейдіть** в Settings → Pages
3. **Виберіть** branch `main` і папку `/root`
4. **Ваш сайт буде доступний** за адресою `https://ваш-юзер.github.io/bukhgalteriya`

## 📋 Тестування (пошагово)

### 1. Реєстрація
- Перейдіть на `register.html` або натисніть кнопку "Реєстрація"
- Введіть нікнейм, email, пароль
- Нікнейм мінімум 3 символи, пароль мінімум 6 символів
- Натисніть "Створити акаунт"

**Тестові дані для першого акаунту:**
```
Нікнейм: testuser1
Email: test@example.com
Пароль: 123456
```

### 2. Вхід
- Перейдіть на `login.html`
- Введіть нікнейм та пароль
- Опціонально: галочка "Запам'ятати мене" (збережеться в cookies)
- Натисніть "Увійти"

### 3. Створення групи
- На дашборді натисніть "➕ Нова група"
- Введіть назву (мін. 2 символи)
- Виберіть тип: 🌐 Публічна або 🔒 Приватна
- Натисніть "Створити"

### 4. Додавання учасника
- Натисніть кнопку "➕ Учасник" на карточці групи
- Введіть нікнейм існуючого користувача
- Натисніть "Додати"
- *Спочатку зареєструйте другого користувача!*

### 5. Створення документа
- Відкрийте групу (натисніть на карточку)
- Натисніть "📝 Новий документ"
- Введіть назву, натисніть "Створити"
- Редактор відкриється автоматично

### 6. Редагування документа
- Скористайтесь **тулбаром** для:
  - **Жирний/Курсив/Підкреслення** — B, I, U
  - **Списки** — маркерований (≡) та нумерований (1.)
  - **Таблиці** — натисніть ⊞ і виберіть розмір
  - **Заголовки** — H1, H2
  - **Вирівнювання** — вліво, по центру, вправо
  - **Колір тексту** — палітра
- **Автозбереження** кожні 10 секунд (або натисніть 💾 Зберегти)
- **Експорт у .txt** — кнопка "Експорт .txt"

### 7. Завантаження файлу
- Натисніть "📁 Завантажити файл"
- Перетягніть файл або виберіть його
- Підтримувані формати: PDF, DOCX, PPTX, XLSX, TXT, PNG, JPG
- Натисніть "Завантажити"
- Файл зберігається як Base64 у localStorage

### 8. Пошук та фільтрація
- Пошук документів по назві (у реальному часі)
- Фільтрація по датам (з - до)
- Кнопка "Очистити" для скидання фільтрів

## 🔐 Безпека та приватність

⚠️ **Важливо!** Це локальне додаток, розроблене для навчання:
- Усі дані зберігаються **в localStorage браузера**
- Паролі **не хешовані криптографічно** (базовий хеш)
- **Не використовуйте реальні паролі!**
- При очищенні кеша браузера — всі дані видаляться

Для продакшену:
- Додайте реальне хеширование паролей (bcrypt, PBKDF2)
- Добавьте сервер (Node.js, Python, тощо)
- Використовуйте HTTPS та защиту сессій

## 🎮 Основні команди дебага

Відкрийте **консоль браузера** (F12) і скопіюйте:

```javascript
// Див усіх користувачів
Storage.getUsers()

// Див усіх груп
Storage.getGroups()

// Див усіх документів
Storage.getDocuments()

// Очистити ВСІ дані (обережно!)
localStorage.clear()

// Отримати поточного користувача
Storage.getCurrentUser()
```

## 🐛 Відомі обмеження

- Максимум ~5 МБ на файл (localStorage обмеження)
- Один логін на раз (單сесія)
- Базова валідація на клієнті
- Без синхронізації у реальному часі (для реальної співпраці потрібен сервер)
